### Week 1 Achievements:

* Successfully created and configured an AWS Free Tier account, and learned how to set up AWS Budgets to manage and control costs.
* Mastered the concepts of identity management and security:
  * Access Management with AWS IAM (Users, Groups, Policies).
  * Securely granting application permissions via IAM Roles.

* Understood and manually deployed basic and hybrid network infrastructures:
  * Creating isolated virtual networks with Amazon VPC.
  * Setting up an integrated hybrid DNS system between Local and VPC with Amazon Route53.

* Became familiar with Compute and Storage services:
  * Successfully launched and connected to an Amazon EC2 instance.
  * Optimized compute costs using Amazon Lightsail.
  * Deployed a Static Website Hosting on Amazon S3.

* Understood the operation of Databases and Monitoring tools:
  * Created a relational database using Amazon RDS.
  * Set up system monitoring dashboards with Amazon Cloudwatch.

* Improved automation capabilities and CLI operations:
  * Automated system scaling with Amazon EC2 Autoscaling.
  * Installed and proficiently used AWS CLI to interact with services.
  * Experienced the cloud-based IDE with AWS Cloud9.

* Successfully integrated the above services to complete the Highly Available Web Application Workshop.
