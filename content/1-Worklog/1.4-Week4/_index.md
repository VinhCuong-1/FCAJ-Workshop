### Week 4 Achievements:

* Security and Compliance:
  * Configured centralized access by setting up Single Sign-On (Amazon SSO) for Organization.
  * Enhanced security with strict policies using IAM Permission Boundaries and limiting role transfer by Condition.
  * Ensured compliance and monitored security standards continuously using AWS Security Hub.

* Application and Data Protection:
  * Protected web applications and APIs from common online threats using Web Application Firewall (AWS WAF).
  * Managed and protected sensitive data through encryption with Key Management Service (AWS KMS).

* Reliability and Network Architecture:
  * Ensured system reliability by deploying automated backup plans with AWS Backup.
  * Built interconnected network architectures by linking Virtual Private Clouds (VPCs) with VPC Peering and centrally managing connections with AWS Transit Gateway.
