### Week 9 Achievements:

* **Container Services & CI/CD Pipelines:**
  * Successfully migrated applications from Monolithic to Microservices architecture using Docker and ran serverless deployments on AWS Fargate.
  * Established automated Continuous Integration and Continuous Deployment (CI/CD) pipelines for applications running on Amazon EKS using AWS CodePipeline integrated with GitHub.

* **Data Lake Construction & Data Processing:**
  * Set up a centralized Data Lake storage architecture on Amazon S3 with optimized zone partitioning (Raw, Processed, Curated zones).
  * Successfully configured and loaded custom personal datasets into the Data Lake.

* **Data Analytics & Visualization:**
  * Mastered core AWS analytics services, including Amazon Athena, AWS Glue, and Amazon Redshift.
  * Connected Amazon Athena query results to Amazon QuickSight, designing visual dashboards to track key metrics.

* **Machine Learning with Amazon SageMaker:**
  * Understood the end-to-end Machine Learning lifecycle on AWS using Amazon SageMaker, covering data preparation, model training, and online endpoint deployment.
