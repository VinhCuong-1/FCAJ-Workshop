### Week 3 Achievements:

  * Mastered automation and serverless operations by implementing automated server shutdowns and integrating Slack notifications using AWS Lambda.
  * Implemented robust system monitoring solutions using Amazon CloudWatch integrated with Grafana dashboards.

* Resource Organization and Access Control:
  * Gained expertise in organizing AWS resources logically by utilizing Tags and Resource Groups.
  * Successfully enforced tag-based access control policies via AWS IAM to manage EC2 service access.

* Systems Management and IaC:
  * Automated service management and simplified operational tasks using AWS Systems Manager.
  * Securely managed and connected to EC2 instances using AWS Systems Manager - Session Manager, eliminating the need for traditional SSH.
  * Started building Infrastructure as Code (IaC) by initializing cloud environments with AWS CloudFormation.
